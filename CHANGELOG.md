# Ollama Agent 更新日志

本文档记录了 Ollama Agent 项目的重要更新和功能改进。

## 1.0.2 - 更新

### 新增功能：代码解释器

**核心功能**
- 添加了安全的沙盒代码执行环境
- 支持在对话中编写和执行 Python 代码
- 提供实时代码编辑器和执行结果展示

**技术实现**
- 创建 `sandbox_executor.py` 沙盒执行器
- 实现隔离的代码执行环境，确保系统安全
- 支持标准输出、错误信息和文件下载
- 添加执行时间限制（30秒）和输出大小限制（10MB）

**安全特性**
- 限制访问系统敏感资源
- 安全的模块导入机制
- 防止恶意代码执行
- 自动清理执行环境

**前端集成**
- 在聊天界面添加代码解释器按钮
- 实现代码编辑器面板
- 支持语法高亮和实时执行
- 添加文件下载和结果展示功能

### 新增功能：自动化启动脚本

**核心功能**
- 创建 `launcher.py` 自动化启动器
- 整合所有启动和测试功能
- 提供智能化的项目启动流程

**执行流程**
1. **环境检查阶段**
   - 验证 Python 版本和依赖包
   - 检查 Ollama 服务状态
   - 验证系统环境配置

2. **应用启动阶段**
   - 启动 Flask 应用服务
   - 验证应用运行状态
   - 检查端口可用性

3. **集成测试阶段**
   - 执行完整的 API 测试
   - 验证 Socket.IO 连接
   - 测试代码解释器功能
   - 检查文件上传和下载

4. **条件启动阶段**
   - 仅当测试全部通过后显示启动菜单
   - 提供交互式启动选项
   - 支持多种启动模式

**启动选项**
- `python launcher.py` - 完整流程（推荐）
- `python launcher.py --check-only` - 仅检查环境
- `python launcher.py --test-only` - 仅运行测试
- `python launcher.py --quick` - 快速启动（跳过测试）

**错误处理**
- 详细的失败诊断报告
- 包含系统环境信息
- 完整的错误堆栈跟踪
- 自动保存诊断日志文件

### 新增功能：缓存清理

在启动菜单中添加了清理功能，用于清除项目运行过程中产生的缓存和冗余文件。

**清理范围**
- 清除 `sandbox_output/run_*` 测试执行产生的临时目录
- 清除项目中的 `__pycache__` 目录（Python 字节码缓存）
- 清除独立的 `*.pyc` 编译文件
- 跳过 `.venv` 和 `.git` 目录，确保安全
- 保留 `logs/` 目录中的异常日志文件
- 保留 `uploads/` 目录中的用户文件

**操作方式**
- 启动菜单新增「清理缓存和冗余文件」选项（选项 4）
- 清理完成后自动返回菜单，可继续选择启动或其他操作
- 显示详细的清理统计（删除的目录数、文件数、释放空间）

### 菜单优化
- 启动菜单从 4 项扩展为 5 项
- 添加菜单循环逻辑，清理操作后自动返回菜单
- 优化菜单描述，明确各选项功能
- 退出选项编号从 4 调整为 5
